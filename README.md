# Project Nook API

School project aiming at the creation of a platform for event countdown

## Production site

Deploy with heroku

```In progress```

## Getting Started 

1. Create and fill the `.env` file like the exemple `.env.example`

2. Install depedencies
```npm install```

3. Launch the app
```nodemon server```

### Prerequisites

- NodeJS
- MongoDB database
